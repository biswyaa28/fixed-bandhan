## Ongoing Optimization

### Monthly Tasks

- [ ] Review keyword rankings (use AppFollow or free ASO tools)
- [ ] Update promotional text with new features/milestones
- [ ] Respond to ALL app store reviews within 48 hours
- [ ] Monitor competitor keyword changes
- [ ] Track install conversion rate (impressions → installs)

### Quarterly Tasks

- [ ] A/B test screenshots (Play Store experiments)
- [ ] Update screenshots with new features
- [ ] Refresh description with latest metrics/testimonials
- [ ] Add new keywords based on search trend data
- [ ] Localize for Hindi (full listing translation)
- [ ] Review and update age rating if needed

### Review Management

| Star Rating | Response Strategy |
|------------|-------------------|
| ⭐⭐⭐⭐⭐ | Thank user, ask for referral |
| ⭐⭐⭐⭐ | Thank user, ask what we could improve |
| ⭐⭐⭐ | Acknowledge feedback, explain fix if applicable |
| ⭐⭐ | Apologize, ask for details via support email |
| ⭐ | Apologize, offer to resolve via support@bandhan.ai, fix root cause |

**Rules:**
- Never argue with a reviewer
- Always be empathetic and professional
- Include support email in negative review responses
- Update review response when issue is fixed
- Target: 4.5+ average rating within first 1,000 reviews

### Review Prompting Strategy

- Trigger: After user's first successful match (positive emotional moment)
- Frequency: Maximum once per 90 days per user
- Never prompt after: rejection, limit hit, error, report
- Copy: "Enjoying Bandhan? Your review helps others find their life partner too. ⭐"
- Easy dismiss: "Not now" button, never shows again for 90 days

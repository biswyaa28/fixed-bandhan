- [ ] NPS > 40
- [ ] Day-30 retention > 15%
- [ ] Organic referral rate > 20% of signups
- [ ] Users complete profiles without prompting (>50%)
- [ ] Unprompted positive app store reviews

### Growth Signals
- [ ] 10K MAU achieved
- [ ] Growth rate > 20% month-over-month
- [ ] CAC < ₹250

### Revenue Signals
- [ ] 3%+ free-to-paid conversion
- [ ] MRR > ₹1L
- [ ] Monthly churn < 10%

## If We DON'T Hit Targets

### Scenario: MAU < 5K at Week 12
**Diagnosis:** User acquisition problem
**Action:**
1. Double down on campus program (more colleges, better incentives)
2. Increase content marketing frequency
3. Test paid ads earlier (₹10K budget experiment)
4. Consider city-specific launch events

### Scenario: Conversion < 2% at Week 12
**Diagnosis:** Premium value proposition weak
**Action:**
1. Interview 20 free users — "Why haven't you upgraded?"
2. Test lower price point (₹299/month)
3. Add more premium-exclusive features
4. Test "limited-time offer" pricing

### Scenario: Day-7 retention < 15%
**Diagnosis:** Core product problem
**Action:**
1. User interviews — "Why did you stop using the app?"
2. Analyze drop-off points in the funnel
3. Improve onboarding (reduce time to first match)
4. Add push notification engagement loops
5. Consider pivoting matching approach

### Scenario: NPS < 20
**Diagnosis:** Product doesn't solve the problem well enough
**Action:**
1. Deep user research (10 in-person interviews)
2. Competitive analysis — what are users choosing instead?
3. Consider fundamental product changes
4. Extend runway before raising seed

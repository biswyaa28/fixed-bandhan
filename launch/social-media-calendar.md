## Content Guidelines

### Do
- Share real metrics (builds trust)
- Use bilingual captions (English + Hindi)
- Celebrate users (with permission)
- Be transparent about the journey
- Use the comic book visual style consistently

### Don't
- Never use fake testimonials
- Never show actual user photos without consent
- Never mock competitors by name
- Never make promises about finding a partner
- Never use clickbait or misleading content
